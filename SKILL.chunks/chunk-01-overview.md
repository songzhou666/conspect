# conspect Skill 概览与触发条件

## 产品概述

conspect 是全自动多源数据智能分析与商务报表渲染工具。它能够从 Excel、CSV、JSON 等多种数据源导入数据，自动完成数据清洗、维度识别、指标计算，并根据数据特征智能选型图表，最终生成可直接交付的商务级数据看板。

## 核心约束

| 约束项 | 说明 |
|--------|------|
| 零客户端依赖 | 前端不安装任何插件/运行时，纯浏览器即可查看 |
| 后端全量处理 | 数据清洗、计算、图表组装全在后端完成 |
| ECharts可视化 | 图表渲染统一使用 ECharts 引擎 |
| 数据安全 | 原始明细数据不上前端，仅聚合指标用于渲染 |

## 触发词

以下关键词可激活 conspect Skill：

- **报表类**：生成报表、制作报表、出报表、周报、月报、季报
- **数据分析类**：数据分析、数据看板、BI看板、数据可视化、数据大屏
- **Excel相关**：Excel出图、Excel数据、导入Excel、Sheet处理
- **汇报类**：汇报材料、数据汇报、PPT数据、商务图表
- **输出类**：出图、画图、生成图表、做图表、可视化

## 三种输出形态

| 形态 | 说明 | 适用场景 |
|------|------|----------|
| 交互式Web看板 | 基于 ECharts 构建的完整 HTML 看板，支持筛选/联动/hover/下钻 | 管理层日常监控、数据分析探索 |
| 离线HTML | 单文件轻量化 HTML，内嵌聚合 JSON 数据，不依赖外部服务 | 邮件发送、离线查阅 |
| PDF高清长图 | 通过 Playwright 无头浏览器截图生成的 PDF/高清 PNG | 正式汇报、打印归档 |

## 核心亮点

| 亮点 | 说明 |
|------|------|
| 无客户端依赖 | 只需浏览器即可查看，无需安装任何软件 |
| AI自动搭报表 | 智能识别维度与指标，自动选型图表，自动布局 |
| 大数据量不卡顿 | 分页加载 + 聚合计算，百万级数据流畅渲染 |
| 多源数据融合 | 支持多文件、多 Sheet 批量导入，自动关联 |
| 商务级视觉规范 | 浅色办公风，低饱和配色，清晰的信息层级 |
